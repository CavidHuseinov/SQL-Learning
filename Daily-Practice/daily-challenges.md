# Daily Challenges

Mixed exercises combining multiple SQL concepts for comprehensive practice.

## Challenge 1: Customer Analysis (Fundamentals)
Write queries to answer these questions:
1. List all unique cities where customers live, sorted alphabetically
2. Count how many customers are from each country
3. Find all customers whose names start with 'J' or 'S'
4. Show customers from USA or UK, ordered by registration date

## Challenge 2: Product Insights (Aggregation)
1. Find the most expensive product in each category
2. Calculate the total inventory value for all products
3. Show categories with average price above $200
4. Count products in each price range: <$100, $100-$500, >$500

## Challenge 3: Order Analysis (JOINs)
1. List all orders with customer names and email addresses
2. Find customers who have never placed an order
3. Show total number of orders per customer, including customers with 0 orders
4. Display order details with product names for all completed orders

## Challenge 4: Sales Performance (Complex Queries)
1. Calculate total revenue by product category
2. Find the top 5 customers by total amount spent
3. Show products that have been ordered more than 3 times
4. Calculate running total of sales by order date

## Challenge 5: Employee Analysis (Subqueries)
1. Find employees earning more than the company average
2. List employees with salary higher than their department average
3. Show employees who have direct reports (are managers)
4. Find the highest paid employee in each department

## Challenge 6: Advanced Scenarios (All Concepts)
1. Create a report showing:
   - Customer name
   - Number of orders
   - Total spent
   - Average order value
   - Only for customers with more than 1 order
   - Sorted by total spent (highest first)

2. Analyze product performance:
   - Product name and category
   - Total quantity sold
   - Total revenue
   - Average price per unit
   - Rank within category by revenue

3. Department salary analysis:
   - Department name
   - Number of employees
   - Average salary
   - Salary range (min to max)
   - Percentage of total company salary budget

## Weekly Challenge: Build a Complete Report

Create a comprehensive business intelligence report that includes:
- Customer acquisition trends by month
- Product sales trends
- Revenue by category
- Top performing products
- Customer lifetime value analysis
- Employee productivity metrics

Use CTEs, window functions, and multiple JOINs to create this report!

**Pro Tip**: Try solving these without looking at solutions first. The struggle is where the learning happens!
